## 规则说明

| 类型 | 说明                       | 备注                                       |
| :--- | :------------------------- | :----------------------------------------- |
|      | 开头是大写的英文`C`或者`W` | C表示电子版的，也就是电子卡。W表示办的本子 |
|      | 后面是8位数字              |                                            |
