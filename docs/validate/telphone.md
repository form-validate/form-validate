## 代码
``` js
/^(0\d{2,3})?(-)?\d{7,8}(-\d{1,6})?$/
```

## 规则说明
::: tip
格式:`区号-号码-分机号`
:::

| 类型   | 说明                                        | 备注 |
| :----- | :------------------------------------------ | :--- |
| 区号   | 区号以0开头，3位或4位                       |      |
| 号码   | 号码由7位或8位数字组成                      | --   |
| 分机号 | 分机号码由1-6位数字组成                     | --   |
|        | 区号与号码之间可以无连接符，也可以“-”连接 | --   |
|        | 分机号可有可无                              | --   |
