## 规则说明
:::tip 说明
格式：`年-月-日 时:分:秒`
:::
| 类型 | 说明                       | 备注                        |
| :--- | :------------------------- | :-------------------------- |
| -    | `年-月-日 时:分:秒`        | 例如：`2017-02-28 11:19:20` |
| -    | 月只能是从01-12            |                             |
| -    | 日只能从01-31              |                             |
| -    | 匹配出二月份（28天或29天） |                             |
| -    | 小时支持12小时制和24小时制 |                             |
| -    | 分钟00-59                  |                             |
| -    | 秒00-59                    |                             |
