## 规则说明
::: tip
所有邮箱都符合`【名称@域名】`的规律
:::

| 类型 | 说明                                   | 备注         |
| :--- | :------------------------------------- | :----------- |
| 名称 | 由26个英文字母、0-9、`-`、`_`、`.`组成 | 不区分大小写 |
|      | `-`、`_`、`.`不能用作开头和结尾        | --           |
|      | 最长为64个字符                         | --           |
| 域名 | [域名验证](/validate/domain.md)        | --           |

## 常见组合
| 类型         | 示例           | 备注 |
| :----------- | :------------- | :--- |
| 纯数字       | 123456@qq.com  | -    |
| 纯字母       | abc@qq.com     | -    |
| 字母数字混合 | abc123@qq.com  | -    |
| 带点的       | abc.123@qq.com | -    |
| 带下划线     | abc_123@qq.com | -    |
| 带连接线     | abc-123@qq.com | -    |


## 邮箱@后缀的类型

| 类型     | 示例                  | 备注 |
| -------- | --------------------- | ---- |
| 一级域名 | 123456@qq.com         | -    |
| 二级域名 | 123456@vip.qq.com     | -    |
| 三级域名 | 123456@abc.vip.qq.com | -    |

邮箱还可以是中文域名的，[http://互联网.中国/index.jsp](http://互联网.中国/index.jsp)，例如：`张三@互联网.中国`
