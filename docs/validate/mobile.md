## 号码组成
目前我国使用的手机号码为11位，其中各段有不同的编码方向

前3位：网络识别号

第4～7位：地区编码

第8～11位：用户号码

电话号码编码规则：国家代码 + 地区代码 + 电话号码（服务商代码 + 4位用户号码）
