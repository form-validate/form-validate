# 代码

  /^((1[45]\d{7})|([G|g]\d{8})|([P|p]\d{7})|([S|s]\d{7,8})|([D|d]\d+)|([E|e]\d{8}))$/

# 正则分析

在线分析-中国护照

# 规则说明

类型 说明 备注
P开头+7位数 因公普通
D开头 D=diplomatic 外交护照
E开头+8位 有电子芯片的普通护照
S开头+7位数 或 +8位数 公务
G开头+8位数 因私普通护照
14开头+7位数 因私普通护照
15开头+7位数 因私普通护照
开头字母大小写都可以
